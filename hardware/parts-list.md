# 4DX@HOME 部品リスト・調達情報

## 1. 部品リスト概要

### 1.1 プロジェクト必要部品
4DX@HOMEプロジェクトの完全な実装には、以下のカテゴリの部品が必要です：

- **コンピューティング**: ラズパイ、Arduino等の制御基板
- **アクチュエーター**: 振動モーター、香り拡散装置等の体感デバイス
- **電子部品**: 制御IC、抵抗、コンデンサ等の回路部品
- **機械部品**: 筐体、マウント、ケーブル等の構造部品
- **電源**: 各種電源アダプター、バッテリー

### 1.2 コスト概算
- **最小構成（振動のみ）**: 約 ¥16,000
- **標準構成（振動+香り）**: 約 ¥25,000  
- **フル構成（全体験対応）**: 約 ¥35,000
